Install
------------------------------------------------------------------------------

``my_icon_vault`` is released on PyPI, so all you need is to:

.. code-block:: console

    $ pip install my-icon-vault

To upgrade to latest version:

.. code-block:: console

    $ pip install --upgrade my-icon-vault
